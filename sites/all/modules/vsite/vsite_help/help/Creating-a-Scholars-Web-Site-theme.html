Custom themes for individual scholar sites can be added to individual installations of the project software. Themes consistent of CSS and any custom images, much as one would at the famous CSS Zen Garden site. You do not have access to the markup itself - only the presentation layer. In this way we can provide an elegant and scalable application for all.

Your custom theme will be a sub-theme of out 'base theme' which will also load. Your CSS will override CSS in the base theme. You can create clean and compact CSS by choosing to keep whichever defaults from the base theme you want. We therefore recommend becoming familar with the markup we provide and the default CSS. Much of that can be done in the usual ways (read: Firefox and Firebug..) but here we provide some general background.
